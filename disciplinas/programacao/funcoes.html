Funções em Programação

Programação

Uma função consiste São códigos pré-programados, geralmente definidos antes da função principal. A função recebe um nome e, através desse, ela é chamada. Basicamente, a maioria da linguagem C é composta por funções. As funções são utilizadas para:

  • Permitir o reaproveitamento de código;
  • Evitar a repetição de um trecho de código no main();
  • Facilitar a correção;
  • Facilitar o entendimento, pois com elas os blocos não ficam grandes demais;
  • Separar o programa em partes (blocos); assim, o main() se torna um chamador de funções.

Sintaxe

O bloco de comando que será executado dentro das funções é delimitado por chaves ({}).

<tipo de dado> <nome>(<argumento(s)>, ...){
	// código
	return <retorno>;
}
  • tipo de dado: o tipo de dado que ela retorna ao terminar (void, char, int, float, ...);
  • nome: qualquer nome; define como se chama a função.
  • argumento(s): regras e definições atribuídos ao(s) dado(s) recebido(s) pela função. Uma função pode ter quantos argumentos forem necessários.
  • retorno: definido pelo comando return, faz com que a função retorne um dado como resultado da operação (exceto funções do tipo void) ao seu término; geralmente, é a última linha do bloco de código.

Considerações

  • Para cada função criada, devemos ter pelo menos uma chamada da função no programa;
  • Uma função pode ter quantos argumentos forem necessários, mas a chamada da função deve ter o mesmo número de parâmetros, todos respectivamente na mesma ordem e do mesmo tipo;
  • Cada função só pode retornar um valor e a variável que o recebe deve ser do mesmo tipo do que o retorno;
  • Uma função pode ser declarada antes ou depois da função principal. Se for declarada depois: antes do main() deve ser declarado seu protótipo;
  • Uma função pode chamar outra função e, desta forma, a função chamada deve ser declarada antes da chamadora;
  • As funções podem ser declaradas dentro de um arquivo de cabeçalho do usuário, mas devem ser observados os itens acima.
