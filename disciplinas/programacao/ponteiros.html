Ponteiros

Programação

O Ponteiro é uma variável que guarda o endereço (&) de outra variável. Portanto, para que serve?

  • Evitar a utilização de variáveis globais;
  • Torna o código mais independente (variáveis locais), melhor para fazer manutenção;
  • Aproveitar espaços livres da memória; economiza, racionaliza o uso da memória;
  • Ao passar valores pra uma função, eu manipulo uma cópia da variável. Com ponteiros, eu posso manipular a própria variável.
<tipo de dado> *<nome>;
//Ex:
int *ptr, *pt, *p;

Operadores com Ponteiros

Endereço (&)

&<var>; //indica o endereço da memória do seu operando 'var'.
//Ex:
int *p, x, y;
p = &x;

Conteúdo (*)

<var> = *<ptr>; //a variável 'var' terá o conteúdo do ponteiro.
//Ex:
int *p, x, y;
p = &x; // inicialização do ponteiro.
y = *p;

Agora y é igual ao conteúdo do ponteiro p, que aponta para o endereço da variável x; ou seja, y = x.

Incremento (++)

int *p, x, y;
p = &x;
p++;
*p = y; //p = &(x+4), x+4 = ???

Um ponteiro de um tipo de dado específico, ao ser incrementado, vai avançar a quantidade de endereços, em bytes, necessários para guardar aquele tipo de variável.

char *char_p;
char_p++; //O ponteiro 'char_p' avançará 1 endereço = 1 byte.

int *int_p;
int_p = &<var>;
int_p++; //avançará a quantidade necessária para guardar uma variável do tipo int (4 bytes = avança 4 endereços: int_p+=4)
