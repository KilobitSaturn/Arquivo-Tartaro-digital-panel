Matrizes

As matrizes, em programação, são um conjunto ordenado de dados de enésimas dimensões, ou seja, de quantas dimensões forem necessárias. Uma dimensão é definida usando um número natural dentro de colchetes. Na linguaguem C, todas as enumerações começam com 0. Por exemplo: em [10], as posições vão de 0 a 9.

<tipo de dado> <nome da matriz>[m][n][p][q][...];

Exemplos

int mat[m][n];
  • int m: número de linhas;
  • int n: número de colunas;
#define QNT
#define CAM
#define LIN
#define COL
    
float mat1[CAM][LIN][COL];
int mat2[QNT][CAM][LIN][COL];

Vetores

Vetores são matrizes de apenas 1 dimensão.

int vet[v];
float func[f];
double doll[d];

Strings

As strings são um caso específico dos vetores: são vetores char, apenas constituido de caracteres. Toda string termina em '\0' (indica final de uma string), o que ocupa um caractere do vetor. Toda vez que for trabalhares com vetores, tome cuidado para não sobrepor outros vetores/memórias/dados. Você é o único que sabe a quantidade de dados a serem usados.

 char str[c];
