A linguagem “C” surgiu de uma implementação da “B”, que por si só era composta de funções de baixo nível, ou seja, que mexiam diretamente com os valores e endereços de memória do dispositivo. Projetada em 1972, no laboratório da Bell, por Dennis Ritchie, a “C” é tida por uma linguagem de médio nível, pois implementa instruções a nível de bit e endereços (o que só era feito através do assembler, um tipo de compilador) e instruções de alto nível (o que ocorre na maioria das linguagens de programação).

As qualidades da Linguagem C

Portabilidades

comitê ANSI (American National Standards Institute - equivalente à ABNT no Brasil) iniciou um trabalho de padronização que foi seguido, em grande parte, pelas implementações comerciais. O C iniciou e popularizou esse padrão das linguagem de programação.

Disposições

  • Conjunto básico de tipo de dados;
  • Instruções para programação estruturada;
  • Possui um pequeno número de comandos;
  • Protótipos de função;
  • Possui ponteiros e aritmética de endereços.

Estrutura de um programa em C

Todo programa escrito em linguagem C necessita de uma estrutura mínima para poder ser compilado e não gerar erro. Como a linguagem tem poucos comandos, a grande maioria das ações são realizadas através das funções.

Desta forma, a primeira declaração deve ser a das bibliotecas de funções (#include), seguindo das declarações (#define) e as variáveis globais (públicas). O próximo passo é declarar as funções e, dentre elas, devemos destacar a obrigatoriedade da função principal chamada main.

O pré-processador

O pré-processador é um programa que examina o programa-fonte em C e executa certas modificações antes da compilação com base em instruções - chamadas de diretivas. - As diretivas do pré-processador devem fazer parte do programa fonte, mas não farão parte do programa executável, pois elas são retiradas pelo pré-processador assim que forem executados e se tornarem desnecessários.

  • #include:

    A diretiva include provoca a inclusão de outro arquivo no programa fonte. O efeito obtido é a apresentação de um texto, como se tivéssemos digitado todo o conteúdo do arquivo e o associado ao programa atual.

  • #define:

    A diretiva define usada para definir constantes com nomes apropriados (na sua forma mais simples). Por convenção, o identificador é escrito em letras maiúsculas e cada diretiva deve estar escrita em uma nova linha.

A ordem dos fatores altera o produto

Não apenas as diretivas, mas todas as linhas de código são lidas consecutiva e respectivamente, de cima para baixo. Isso significa que o resultado das operações lógicas que você colocar podem ser diferentes dependendo da ordem em que elas estão.

Variáveis

As variáveis são a forma mais direta de guardar dados dentro da memória de um dispositivo eletrônico. Todas as variáveis são compostas por elementos que devem ser decididos pelo programador (você!), com exceção do endereço de memória:

  • Um tipo de dado: podem ser qualquer um dos tipos de dados citados no próximo subtítulo.
  • Um identificador: qualquer nome que contenha letras maiúsculas e minúsculas, números e subtraço (_); desde que não conflitam com os identificadores de outros elementos do sistema.

    Tamanho é documento
    A linguagem C é "case sensitive", isto é: letras maiúsculas ou minúsculas são interpretadas de maneiras diferentes. Se você declarar uma variável com o nome soma ela será diferente de Soma, SOMA, SoMa ou sOmA. Da mesma maneira, os comandos do C só podem ser escritos em letras minúsculas pois, senão, o compilador não irá interpretá-los como sendo comandos.

  • Um endereço de memória: é um número que indica onde se encontra essa variável na memória eletrônica - o endereço é escolhido automaticamente pelo algoritmo de memória.
  • Um local: dependendo de onde você declara as suas variáveis, elas podem ser acessadas por todo o seu programa ou apenas seções específicas dele. Abordaremos isso posteriormente.

    • Tipos de dados

    Na linguagem C, existem 5 tipos básicos de dados. O limite do alcançe - o quão longínquos são os valores que podem ser armazenados em certo tipo de dado - são definidos pelo tamanho de armazenamento necessário para guardá-lo. Todos os tipos de dados possuem um tamanho fixo.

Identificador Tamanho (Bytes) Descrição
void 0 Vazio. Nada.
char 1 Armazena um (e apenas um) caractere conforme a tabela ASCII, possuindo um valor numérico inteiro correspondente.
int 2 ou 4 Armazena um valor numérico inteiro (\(\in \mathbb Z\)).
float 4 Armazena um valor numérico real (\(\in \mathbb R\)).
double 8 Armazena um valor numérico real (\(\in \mathbb R \)) com o dobro de limite.
  • Por que char possui um valor numérico?

    Assim como os int, as variáveis char guardam na memória um valor numérico inteiro; mas, diferente deles, identificar uma variável como char faz com que o algoritmo de leitura converta esse valor numérico em um caractere, um símbolo ou comando. Esse caractere é correspondente com o Código Padrão Americano para o Intercâmbio de Informação (do inglês: “American Standard Code for Information Interchange” ou ASCII, pronunciado /’áski’/), que é o mesmo usado pelo seu teclado.

Operações das Variáveis

Declaração

Declarar uma variável significa reservar um espaço de memória do dispositivo eletrônico para guardar valores. Em C, elas são feitas da seguinte maneira:

tipo_de_dado identificador; //(o ponto e vírgula é obrigatório no final de cada linha.)
//Exs:
char c;
int info;
float fly;
double dice;

Como você pode ter percebido, void é o único tipo de dado que é indeclarável.

Declarações

Se você precisa de múltiplas variáveis do mesmo tipo de dado, você não precisa declarar todas elas em linhas diferentes você pode fazê-las na mesma linha usando a vírgula (,). Desse modo, você pode declarar quantas variáveis forem necessárias.

int i, j;
char a, b, c;
float pi, phi, e, sqrt2, alpha, delta;

Uma variável declarada mas não atribuída não tem valor definido, ou melhor, ela pode ter qualquer valor que a máquina possa ter descartado na sua própria memória.

Atribuição

Atribuir uma variável significa associá-la a um valor; de forma que, toda vez que essa variável for chamada, ou seja, participar do algoritmo sem ser na sua própria declaração - o que sugere que atribuições só podem ser realizadas em variáveis declaradas -, ela contenha tal dito valor:

<identificador> = <valor>;
/*Exemplos*/
//Ex. 1
int i;
i = 10;
//Ex. 2
float f;
f = 2.754693;
//Ex. 3
double d;
d = 3401695.15342;

A declaração de variáveis char é um pouco mais específica. Você pode atribuir a uma variável do tipo char o valor correspondente a um caractere em ASCII, ou você pode fazer isso diretamente colocando o caractere entre apóstrofos (’).

char c;
c = 99; //99 corresponde ao caractere 'c' em ASCII.
//ou...
c = 'c'; //nessa linha, o caractere 'c' foi atribuído à variável c sem usar o valor seu numérico.
Substituição

As variáveis são chamadas assim porque o seu valor pode ser alterado ou variado de acordo com a necessidade. Substituir o valor de uma variável é um processo simples: é só atribuir outro valor no lugar!

int i; //aqui, a variável "i" não tem valor definido.
i = 10; // aqui, "i" vale 10.
//...   a variável "i" é usada no algoritmo.
i = 2; // aqui, "i" agora vale 2.

Todas as operações realizadas enquanto i valia 10 ainda mantém os seus valores inalterados, apenas as operações realizadas depois que i foi atribuído com 2 usarão esse mesmo valor.

Atribuições

Usar uma constante para atribuir a múltiplas variáveis é possível usando a linguagem C.

int i, j;
i = j = 8; // "i" vale "j" que, por conseguinte, vale 8;

Contração[1]

É possível atribuir um valor a uma variável assim que declará-la no seu código, assim ocorrendo o processo de contração da declaração com a atribuição. Funciona com declaração múltipla também.

char c = 'c';
int i = 10, j = 2, /*...*/;
float pi = 3.1415, phi = 1.6180, e = 2.7183;
double d = 12345.6789;

Funções

A função principal (main)

A função main é única função obrigatória em todos os códigos de programas em C pois ela indica a rotina principal, a mais importante, o que será executado propriamente. O bloco de comando que será executado dentro da função main é delimitado por chaves ({}).

int main{
        //código a ser executado
}

Operadores

Os operadores são usados para executar diversos tipos de operações em C. Os mais comuns são os operadores matemáticos:

Símbolo Operação
+ Adição
- Subtração
* Multiplicação
/ Divisão

Notas

  1. [^] Contração é o processo de unir duas ou mais palavras que, juntas, formam uma nova palavra mais curta, geralmente com a omissão de sons (fonemas).
