Uma variável composta de outras variáveis de tipos distintos

Declaração de uma Estrutura

Diferentemente dos tipos de dados, os tipos de estrutura são personalizáveis, cabendo ao programador escolher quais variáveis estarão dentro deles. É necessário declarar um tipo de estrutura antes de declarar uma estrutura propriamente.

struct </*nome do tipo de estrutura*/> {
	//aqui dentro vão as variáveis que já conhecemos, incluindo ponteiros e matrizes.
	int var;
	char *c, str[50];
	float var1, var2;
	double dd;
	//...
};
  • Todas as regras de declaração de variáveis de qualquer tipo de dado se aplicam aqui.
  • Note que não há como atribuir variáveis na declaração de uma struct, apenas declará-las.

Declaração de uma instância de Estrutura

void main(){
	struct <nome do tipo de estrutura> <nome da estrutura1>, <nome da estrutura2>, ...; //cria duas estruturas distintas do mesmo tipo
	struct <nome do tipo de estrutura> <nome da estrutura>[n]; //cria 'n' vetores de estruturas, cada uma contendo suas respectivas variáveis.
	
//Agora que temos uma estrutura, usamos o separador '.' para acessar suas variáveis:
	<nome da estrutura>.<var> = 0;
}

Exemplos

struct pess {
        int idade;
        float peso;
        char nome[50];
    } pessoa[10]; //declaração abreviada de um vetor de estruturas
struct pess {
        int idade;
        float peso;
        char nome[50];
    };
    
    void main(){
    	struct pess pessoa1, pessoa2; // estruturas distintas
    	struct pess pessoa[10]; // vetor de estruturas
    	pessoa1.idade = 17;
    }

Estruturas Aninhadas

Sigamos o exemplo abaixo, em que há a necessidade de declarar uma estrutura aninhada a outra: 
struct pessoa {
	int idade;
	float altura, peso;
	char nome[60];
};

struct aluno {
	int matricula, turma;
	char turno;
	struct pessoa essecial;
};

// para acessar as variáveis de uma estrutura aninhada em outra, continua-se usando o '.' como separador:

struct aluno aux;
aux.essencial.idade = 18;

Typedef

Colocar typedef antes da definição de uma estrutura omite struct da declaração, pois assim a estrutura age como se fosse um Tipo de Dado

typedef struct srt {
	//...
};

void main(){
	srt s1, s2;
	srt s[10];
}

Ponteiros para estruturas

typedef struct srt{
	int temp;
};

void f(srt *p){
	//para acessar a variável 'temp' usando o ponteiro 'p', usa-se:
	p->temp; // esse formato
	(*p).temp; //ou esse formato (incomum)
}

//ou seja, de forma genérica podemos definir como:
<p_srt>-><var>;
