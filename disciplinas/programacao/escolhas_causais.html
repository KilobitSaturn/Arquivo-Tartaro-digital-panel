switch(<var>){
	case '1':
		// código caso var seja igual ao caractere 1, e não ao valor 1;
		break;
	case 2:case 3:case 4:
		// código caso var = 2, var = 3 e var = 4
		break;
	default:
		// caso aplicado quando <var> não atende os outros casos.
}
  • Você pode fazer casos na função switch usando atribuições, por exemplo: switch(var + 1), switch(var*10), switch(var%2); e construir os casos baseado nos resultados dessas operações.
  • A função switch suporta somente uma variável por vez.
  • O caso default não é obrigatório.
  • Caso você não digite a função break no final de cada caso (exceção de default), todos os casos que forem menores ou iguais ao resultado serão realizados.
