Noções de Arquivos

A linguagem C não contém nenhum comando de Entrada/Saída. Ao contrário, todas as operações de E/S ocorrem mediante chamadas de biblioteca C padrão. O sistema de E/S de C é único, porque dados podem ser transferidos na sua representação binária interna ou em um formato de texto legível por humanos. Essa abordagem faz o sistema de arquivos de C extremamente poderoso e flexível.

Antes de começar a falar do sistema de arquivos C ANSI, é importante entender a diferença entre os termos fluxos e arquivos. O sistema de arquivos de C é projetado para trabalhar com uma ampla variedade de dispositivos, incluindo terminais, acionadores de disco e acionadores de fita. Embora cada um dos dispositivos seja muito diferente, o sistema de arquivo com buffer transforma-os em um dispositivo lógico chamado de stream ou fluxo.

Fluxos

O fluxo é uma abstração do dispositivo real chamado de arquivo. Pelo fato dos fluxos serem amplamente independentes do dispositivo, a mesma função pode escrever em um arquivo em disco ou em algum outro dispositivo, como o console. Existem dois tipos de fluxos:

  1. Texto: Um fluxo de texto é uma sequência de caracteres.
    • O padrão C ANSI permite (mas não exige) que um fluxo de texto seja organizada em linhas terminadas por um caractere de nova linha (`\n`);
    • Porém, o caractere de nova linha é opcional na última linha e é determinado pela implementação;
  2. Binário:Um fluxo binário é uma sequência de bytes com uma correspondência de um para com aqueles encontrados no dispositivo externo – isto é, não ocorre nenhuma tradução de caracteres

Ponteiros de Arquivo

Um ponteiro de arquivo é um ponteiro para uma área na memória (buffer) onde estão contidos vários dados sobre o arquivo a ler ou escrever, tais como o nome do arquivo, estado e posição corrente. O buffer apontado pelo ponteiro de arquivo é a área intermediária entre o arquivo no disco e o programa. Este buffer intermediário entre arquivo e programa é chamado fluxo, e um fluxo é uma entidade lógica genérica, que pode estar associada a uma unidade de fita magnética, um disco, uma porta serial, etc.

Um ponteiro de arquivo é uma variável-ponteiro do tipo FILE que é definida em stdio.h. Para ler ou escrever em um arquivo de disco, o programa deve declarar uma variável ponteiro de arquivo, similar as demais variáveis do programa:

FILE *fp;
  • fp: é o nome que escolhemos para o ponteiro de arquivo.
