O formato do arquivo de imagem BITMAP (.bmp)

Programação

O tipo de arquivo de imagem BITMAP tem suas dimensões avaliadas diferentemente das outras. O dimensionamento da imagem começa no canto inferior esquerdo e segue pelas linhas, em pixels, até o canto superior direito. Para que o programa consiga ler corretamente, é necessário a adição de "Padding" - valores vazios que servem para complementar a linha de pixels - de forma que ela contenha um número de bytes que seja múltiplo de 4. Essa padronização é necessária para o alinhamento correto da leitura dos dados de imagem ao ser lida pelo programa de exibição de imagem.

Descrição do Cabeçalho

Cabeçalho do Arquivo

  • Signature: Contém a palavra "BM".
  • FileSize: Tamanho do Arquivo (em Bytes).
  • Reserved: Reservado, inutilizado.
  • DataOffset: Deslocamento do início do arquivo até o início dos dados do bitmap (em bytes).

Cabeçalho de Informações

  • Size: Tamanho do Cabeçalho de Informações.
  • Width: Comprimento bitmap (em pixels).
  • Height: Altura bitmap (em pixels).
  • Planes: número de planos (geralmente é 1). Outros valores podem indicar a separação cromática em diferentes planos da imagem.
  • BitsPerPixel: é usado para armazenar informações de entrada da paleta. Isso também identifica, indiretamente, o número de cores possíveis. Os valores possíveis são:
    • 1 = paleta monocromática, NumColors = 1
    • 4 = paleta de 4 bits, NumColors = 16
    • 8 = paleta de 8 bits, NumColors = 256
    • 16 = RGB de 16 bits, NumColors = 65536
    • 24 = RGB de 24 bits, NumColors = 16M
  • Compression: Tipo de compressão. Qualquer valor diferente de 0 indica alguma forma de compressão.
  • ImageSize: Tamanho da imagem (comprimida). É válido definir este valor como 0 se a compressão for igual a 0.
  • XpixelsperM: resolução horizontal (px/m).
  • YpixelsperM: resolução vertical (px/m).
  • ColorsUsed: Número de cores efetivamente utilizadas. Para um bitmap de 8 bits/pixel, esse valor será de até 256.
  • ImportantColors: Número de cores importantes. 0 indica que todas as cores são importantes.
