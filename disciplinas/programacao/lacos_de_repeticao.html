Laços de Repetição

Programação

Os laços de repetição são usados para evitar a reiteração de trechos de códigos ou para atenção a uma condição específica que é obrigatória para o avanço ou progresso de futuras operações. Lembre-se de que os comandos dentro dos seus campos de código usam ; pois seguem as regras de código padrão.

Laço for

O uso desse laço é bem restrito, geralmente usado para evitar repetições de operações rotineiras em certos trechos do seu código por se tratar de uma quantidade finita, definida de vezes de repetição. 

stateDiagram
	direction LR
	[*] --> Início
	Início --> Condição
	Condição --> Exceução: Verdadeiro
	Condição --> [*]: Falso
	Exceução --> Condição: Incremento
        
for(<início>; <condição>; <incremento>){
	//código a ser repetido
}

Laço While

O laço da função while é usado para uma quantidade indefinida de repetições de vezes, pois deixará de repetir somente quando a sua condição deixar de ser atendida.

stateDiagram
	direction LR
  	[*] --> condição
	condição --> execução: Verdadeiro
	condição --> [*]: Falso
	execução --> condição
    	
while(<condição>){
	//execução
}

Laço do-while

O seu uso é semelhante ao do laço anterior, com a diferença de que a condição é realizada no fim do bloco de código, antes de retornar ao início do laço. Isso faz com que o código dentro do bloco seja realizado pelo menos uma vez. Esse é o único laço que usa ;.

stateDiagram
	direction LR
  	[*] --> Execução
	Execução --> Condição
	Condição --> Execução: Verdadeiro
	Condição --> [*]: Falso
        
do {
	//execução
} while (<condição>);
